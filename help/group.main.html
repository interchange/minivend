Group Manager


Tallyman allows site managers to place and rearrange pages and products as whole entities, with their formatting, images, and text together. The group manager allows an administrator to mangage other groups of elements in the same manner, such as headers and footers, navigation bars, and lists of links.

Create new group
Select the group type you are creating from the pull-down menu, then click "create new group" to create it. You can make group types in the type manager.

Edit group
