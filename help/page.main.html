Pages


Create New Page
When you create a new page, first specify which type of page it will be in the pull-down menu next to the "create new page" button. This will determine the fields that show up when you make the page. These page types can be configured using the type manager.

If you do not see a pull-down menu with page types next to the "create new page" button, then there is only one page type set up for your store. By default, there should be several page types set up, so the administrator who configured Interchange probably made the decision to remove extra page types. If there is only one page type available, then creating pages will be even easier.

If you have questions about the specific page types listed in the pull-down menu, consult the list of default page types. If the types you see do not match the defaults, ask the administrator who configured Interchange for you.

Edit page
Select a page from the list before pushing this button. The screen will look the same as the create new page, but all the fields will be filled out and ready for you to edit. For the changes to be recorded, you must click on the "OK" button at the bottom of the screen before leaving. Using the browser's back button has the same effect as pushing the cancel button to exit the screen -- no changes will be recorded.

Delete page
Select a page from the list that you would like to delete, and click on "Delete Page". You will be asked for confirmation. If the choice is confirmed the item will be deleted -- permanently.
