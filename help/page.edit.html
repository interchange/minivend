Create New/Edit Page


Keep in mind that only the default fields will be described in this document. You or your administrator can easily create custom fields for particular page types, which will not be described here. Listed below are a few of the default fields, some of which may have been disabled or deleted by the administrator.

Name: The name of the page, which will appear in the list of pages to be edited, rearranged, and managed. It is possible to create two pages with the same name, but it's not recommended.

Title: Often identical to the page name, the title will appear on the web site itself. Where it will appear will depend on the template, but usually it appears near the top of the page and at the top of the browser window.

Content: Text that appears on the page, usually at the top. For some information-only pages, the page content will take up most of the page, but for product pages, the content is often just a short introduction to the types of products that are displayed. If the products you are going to assign to this page require no introduction, you can leave the content field blank.

Filename: Web pages are simply files stored on a web server, identical to some types of files stored on your personal computer. Tallyman lets you specify a file name, for example, you could call your page of beanie babies "beanie.html" or "beanie.babies.html." You must end each filename with ".html" and the filename can not contain any spaces or special characters. Filenames can contain letters, numbers, periods "." underscores "_" or dashes "-".
