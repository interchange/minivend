Create New/Edit Item Fields


Keep in mind that only the default fields will be described in this document. You or your administrator can easily create custom fields for particular item types, which will not be described here. Listed below are a few of the default fields, some of which may have been disabled or deleted by the administrator.

Keep in mind that with the exception of "Name," all fields may be left blank, depending on what type of item it is. Individual page templates may display items differently depending on what is left blank.

[coming soon: list of fields for default item types]

Name: The name of the item, which will appear in the list of items to be edited, rearranged, and managed. It is possible to create two items with the same name, but it's not recommended.

Image: This is the filename of the image, in gif or jpeg format. You will need to seperately upload the images to the server using FTP software.

Description: Text that appears on the page, usually next to the item's name and picture. If the item you are creating requires no description, you can leave the field blank.

SKU: Stock keeping unit or item stock number.

Price: When entering price, it is not necessary to enter a "$" or other currency symbol or a decimal point if the price is a whole number. If the item costs $8.00, entering "8" in the price field is sufficient.

Weight: Entering a weight for each item is only necessary if you are calculating shipping by weight. If you are calculating shipping with a flat rate or based on price, you would not need to enter a weight.
