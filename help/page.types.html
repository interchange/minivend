Page Types


A page type is very similar in concept to an item type. Interchange-generated pages divide content on the page into editable fields, like page title, main page text, and background color. In order to create a new field for a page type, you'll generally have to edit the page template, so this feature is not quite as accessible as the item editor is (that is to say, while a product manager would commonly use the item type manager in the course of day-to-day operations, the page type manager would only be used by a designer who is working on new templates). Each page type has a particular template assigned to it.
