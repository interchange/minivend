Knar Editor


Knar stands for "Knar's Not A Registry." It's kind of like a registry.

The KNAR table stores store-wide configuration information. If you don't about store-wide configuration, it probably won't help you much. On the other hand, if you just can't find the place to tweak the value you're looking for, it might be worth poking around (gently!) in here.
