What is a Matrix?
A matrix is a table of all the possible combinations of options available for an item. For example, a golf club can have options for mens and ladies, right and left handed, several shaft types and flexes, and different lofts, or head angles. But not all lofts are available in left handed, and some aren't available in ladies, and graphite shafts cost extra and all have different SKUs. The matrix contains all the variables. A simple matrix might be a size matrix, which has small, medium, large, and extra large. You would then be able to apply a unique sku or price or inventory information to each variation of each item.

A matrix can be specified for a particular item in the Item Editor, but each matrix must first be created in the matrix editor.

Create new matrix
Here, give your new matrix a descriptive name, such as "drivers and fairway woods" or "build-to-order bikes."

Edit matrix
This feature allows you to edit the name. To edit the matrix's dimentions, use the "show dimentions" feature.

Show dimentions
This feature allows you to create new dimentions and create ranges underneath those dimentions.

Create new dimension
Like the "create new matrix" feature, "create new dimension" simply lets you specify a name for the dimension. Each matrix can have an unlimited number of dimensions.

Example: the "drivers and fairway woods" matrix will have dimensions for shaft type, flex, right/left handed, and loft.

Edit dimension
This feature allows you to edit the name.

Show ranges
This feature will take you to another screen where you can create and edit ranges under that dimension.

Create new range
This will allow you to give your new range a name. Each dimension can have an unlimited number of ranges.

Example: The "flex" dimension of the "drivers and fairway woods" matrix will have range entries for light, firm, and stiff.
