Using the Interchange Administration Site

Some shortcut functions are accessible on each page of the Interchange Administration Site. The following is a list of these functions.

  • Akopia
    The Akopia link is located in the upper left-hand and lower right-hand section of each page. Clicking this button takes you to the Akopia web site.

  • About
    The About link is located in the upper right-hand section of each page. Clicking this button takes you to the Akopia web site product page. The product page displays the most current product information.

  • Interchange
    The Interchange link is located in the bottom left-hand section of each page. Clicking this link takes you to the Akopia Developer's Resource Site. The Developer's Resource Site displays up to date product information, documentation, mailing lists, Akopia services, and more.

  • Store ID
    Your Store ID is located in the bottom middle section of each page. Clicking this link takes you to your store's web site. This allows you to instantly view site changes implemented through the Administration Site.

  • Tables
    The Tables link is located in the lower right-hand section of each page. Clicking this link takes you to the Database Tables page of the Interchange Administration Site. This shortcut function allows you to quickly access table information.

  • Quicklinks
    The Quicklinks link is located in the lower right-hand section of each page. Clicking this link displays a window containing shortcut links to commonly used pages in the Interchange Administration Site.

  • Apply Changes
    The Apply Changes link is located in the lower right-hand section of each page. Clicking this link updates the Interchange program with your changes.

  • Log out
    The Log out link is located in the lower right-hand section of each page. Clicking this link allows you to log out of the Administration site and log back in as a different user.
