MiniVend Tech Tip No. 3

If you don't like the setup of this demo, you can change it in any way you want! Use frames, no frames, imagemaps, or no imagemaps. Very graphical or text-oriented -- It is up to you. The documentation should point you in the right direction.

If you do like the setup of this demo, feel free to cut and paste whatever you like into it. The frame-based demo is packed with about every feature that MiniVend has, and can be used to see how they can be employed.
